## 📋 项目简介

一个基于 Playwright 的学习项目，用于研究：

- **Next.js SSR 渲染机制** — 学习 `__NEXT_DATA__` 的数据结构和页面渲染流程
- **Playwright 自动化** — 练习浏览器自动化操作、页面导航、元素定位
- **异步编程模式** — asyncio 协程与并发控制的实践
- **数据清洗与结构化** — JSON 数据提取、转换、存储的工程实践

## 🛠️ 技术栈

- **Python 3.x**
- **Playwright** — 浏览器自动化框架
- **asyncio** — 异步并发
- **MySQL**（可选）— 数据持久化

## 🔑 学习要点

### Next.js SSR 数据结构

Next.js 应用在服务端渲染时，会将页面数据序列化到 `<script id="__NEXT_DATA__">` 标签中。本项目通过分析该标签的 JSON 结构，学习 SSR 框架的数据传递机制。

### 数据采集流程设计

项目实现了分阶段采集的架构模式：

1. **品牌列表** — 从聚合页提取品牌索引
2. **车系信息** — 按品牌分组获取车系
3. **列表概览** — 翻页采集概览数据
4. **详情页** — 获取完整车辆参数

每个阶段支持断点续采，学习了进度管理和容错设计。

### OCR 文字识别

集成 EasyOCR 对截图进行文字识别，学习了：
- 图像预处理与区域裁剪优化
- OCR 文本后处理与正则解析
- 多引擎对比（WinOCR / EasyOCR）
